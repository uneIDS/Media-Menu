Hemingway

Description

Hemmingway is a writing app that gives specific feedback on writing style and level. Different from other word processors or editing apps, Hemmingway uses visual cues after assessing your writing for readability, to highlight sections that are hard to read or use passive voice and make suggestions.

How our team uses the tool

Checking course components (assignments, discussions, overviews, etc) for readability

Padlet

Description

Padlet is a collaboration and curation tool -- it can be used to store and organize ideas, including text, images, videos, audio, etc. These boards can then be shared, so multiple users can contribute to one board

How our team uses the tool

Working with SMEs and other collaborators to brainstorm and develop ideas for courses. Some SMEs have experience with padlet already, so they use it as a place to compile their own ideas and share them.

H5P

Description

H5P is an interactive coursebuilding tool that supports content types such as quizzes, interactive videos, and branching scenarios.

How our team uses the tool

H5P enables us to create and embed interactive visual content into our courses. It is a formative tool most often used in ungraded activities to scaffold learning, and support coursework and graded assignments.

Note: H5P does not integrate with our LMS, (Brightspace) so grades cannot be extracted from H5P.
