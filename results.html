Google Suite

A collection of business, productivity, collaboration, and edication software. The primary tools include Drive, Docs, Sheets, Slides, and Forms.

A screen and/or webcam recording software that allows the user to edit, zoom, and add in special elements to their recordings.

Zoom

Video conferencing platform that allows users to host meetings across the world. Also allows for collaboration and recording.

Hemingway

A writing app that gives specific feedback on writing style and level. It uses visual cues after assessing your writing for readability.

Padlet

A collaboration and curation tool that can be used to store, organize, and share ideas on a collaborate board.

Hypothes.is

A collaborative, open-source annotation tool that can be used on websites and PDFs to share annotations between multiple people.

Vyond

A full-bodied animation software that allows for easy editing and configuration of stories into animated segments.

H5P

Web-based elements that add easy interactivity to courses. This includes things like interactive videos, presentations, and learning activities.

Articulate Rise

Course authoring tools that provides multiple methods for sharing information displayed in an interactive and responsive manner.

Camtasia

Video-editing software aimed at being simpler and easier to learn than something like Adobe Premiere Pro while maintaining enough functionality.

Twine

Web-based software designed to create branching scenarios/interactive narratives.

Photoshop

Advanced photo-editing software based on the use of multiple layers.

Premiere Pro

Advanced video editing software based on using multiple tracks. More functional but more difficult than Camtasia.

Illustrator

Advanced vector graphics editing program useful for designing logos, blueprints, infographics, and other images that need to be resolution-independent.
