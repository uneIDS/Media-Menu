Hemmingway

What is it?

Hemmingway is a writing app that gives specific feedback on writing style and level. Different from other word processors or editing apps, Hemmingway uses visual cues after assessing your writing for readability, to highlight sections that are hard to read or use passive voice and make suggestions.

Use Cases:

  • Checking papers/assignments for readability

Padlet

What is it and what does it do?

Padlet is a collaboration and curation tool -- it can be used to store and organize ideas, including text, images, videos, audio, etc. These boards can then be shared, so multiple users can contribute to one board

Examples:

list of examples

Planning Considerations:

  • There are a variety of options available including the ability to comment, grade, and rate posts, allow for anonymous or named commentators, and even a profanity filter.
  • There are a variety of layouts such as: Canvas, Shelf, Map, and Timeline
  • Boards can be created as Templates using the "Remake" feature

Limitations:

  • With the free version you can only have 3 boards to start with
  • Padlet is "working on" their accessibility, but are basically compatible with a screen reader software
  • Freedom of background choice can lead to discracting choices (need to pay attention to accessibility

Hypothes.is

What is it and what does it do?

Hypothes.is is a collaborative, open annotation tool that can be used on websites and PDFs (video annotation should be coming soon). It has an easy to navigate user interface and allows multiple people to comment on/annotate the same piece of information as a part of a larger group (can be public or private).

Use cases:

  • For student and instructor use
  • Different share settings
  • Could be used for small/whole class reading discussions, peer editing, and actual collaborative annotation assignments

Vyond

What is it & what can it do?

Vyond allows users to make professional-looking animated videos to present key concepts.

Use Cases:

When a complex idea can be simplified into chunks, and visual representations are helpful, Vyond might be a good fit. When it is not necessary/appropriate to hire actors to play parts, or when backgrounds/scenarios are not possible to recreate live.

Examples:
  • Using evidence in writing
  • EDU 709 - Welcome
  • SSWO 633 - Visit with Vee

Screencast-o-matic

What is it?

Screen and/or Webcam recording software

What can it do?
  • Records audio, screen, and webcam (if desired)
  • Has annotation tools available if needed
  • Primarily used for lectures
  • Recording lectures
  • (more advanced) editing lectures, zoom, add highlights or simple graphic elements (add annotation after the fact)
Planning Considerations:
  • Some time to learn the software, then mostly recording time
  • IDS can help with edits
  • Schedule ~1 hour training with IDS, more if interested in more advanced editing (unless you already know SCOM)

Limitations:

  • More robust edits may be needed
  • 25 seats
  • Webcam can move around, but if you aren’t aware it may cover content
  • We need SMEs to send SOMREC files if we need to edit webcam placement

Twine

What is it and what can it do?

Twine is a game development software that focuses on creating branching scenarios. It creates an online page that is completely customizable to work with storylines created to allow for interactivity.

Examples:
  • SSWO 565
  • Creatures Such as We
  • September 7, 2020
Planning Considerations:
  • Limited user interface that requires more technical coding knowledge
  • Story formatting and level of engagement is of particluar importance
  • Requires more technical support
  • Level of formatting/interactability is limited to how much work you want to put into it
